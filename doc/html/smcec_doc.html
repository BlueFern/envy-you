Right hand side function

The RHS function for the SMC/EC is of the following form

function du = rhs(self, t, u, R, h, K_p)

This form shows which model components are required to couple with the SMC/EC model. These components are generated by calling the shared method of the other model components. Currently these are K_p from the Astrocyte and R, h from the WallMechanics model.

Sharing variables

The shared method generates the coupling variables that need to be sent to other components of the NVU model. The function form is

   function [J_KIR_i, Ca_i] = shared(self, ~, u)

If additional variables are required to pass to other model components, this is the method that you will need to edit.
